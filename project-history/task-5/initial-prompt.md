
Существует api эндпоинт для создания статей (POST /api/news).
При получении запроса необходимо генерировать изображение для статьи с помощью Gemini 2.5 Flash Image, в качестве промпта использовать заголовок из переменной title.
API ключ от Google AI Studio находится в .env файле, в переменной GEMINI_API_KEY.
